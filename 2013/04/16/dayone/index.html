Day One 排版 —— Markdown

摸索了一会儿Day One的排版，最后发现是Markdown语法。

顺便复习了Markdown的用法，总结一下。

Markdown是一种转化文本的轻量级工具，兼顾可读性和易用性。

1. 标题

代码：

# Title1
## Title2
### Title3
#### Title4
##### Title5
###### Title6

效果：

Title1

Title2

Title3

Title4

Title5
Title6

2. 引用

代码：

> Reference1
> > Reference2
> > > Reference3

效果：

Reference1

Reference2

Reference3

3. 列表

代码：

+ Row1
- Row2
* Row3
1. Row4
2. Row5

效果：

  • Row1
  • Row2
  • Row3
  1. Row4
  2. Row5

4. 代码

代码：

`Code1`

    Code2

效果：

Code1

Code2

5. 链接与图片

代码：

[WeiPhone](http://bbs.weiphone.com "威锋网")

[WPER][ref]
[ref]: http://bbs.wper.com (威智网)

![WeiPhone](http://bbsimages.weiphone.com/images/weiphone/logo.png "威锋论坛")

![WPER][pic]
[pic]: http://bbs.wper.com/static/image/wper/logo.png "威智网"

<img src="http://bbsimages.weiphone.com/images/weiphone/logo.png" alt="BBS" title="威锋网" width="200"/ >

效果：

WeiPhone

WPER

6. 强调

代码：

*MarkDown*
_MarkDown_
**MarkDown**
__MarkDown__

效果：

MarkDown
MarkDown
MarkDown
MarkDown

7. 转义字符

代码：

\\ \` \* \_ \{ \} \[ \] \( \) \# \+ \- \. \!

效果：

\ ` * _ { } [ ] ( ) # + - . !
